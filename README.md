# Sidebar Colors

A focused WordPress plugin for color controls in the block editor sidebar.

## Requirements

- WordPress 6.7 or newer
- PHP 8.2 or newer
- Node.js 24
- pnpm 10.28.2
- Composer 2

## Development

```sh
composer install
corepack enable
pnpm install
pnpm start
```

Run `pnpm build` for a production build and `pnpm lint` to check PHP and JavaScript.
